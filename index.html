Welcome to the Dayton Grocer App!

  •  About the project
  •  Contact us
  •  Disclaimer

Grocer provides free information regarding the location of fresh food (organic and unprocessed) in the Dayton Metropolitan area. The store types are indicated by appropriate icons.

This application is created by Code for Dayton volunteers. Code for Dayton provides open source solutions and is a Code for America Brigade.

Features
  • Fullscreen mobile-friendly map template with responsive navbar and modal placeholders
  • jQuery loading of external GeoJSON files
  • Elegant client-side multi-layer feature search with autocomplete using typeahead.js
  • Integration of Bootstrap tables into Leaflet popups
  • Logic for minimizing layer control and switching to modal popups on small screens
  • Responsive sidebar functionality via the leaflet-sidebar plugin
  • Marker icons included in layer control

The data provided on this site is for informational and planning purposes only.
